# Initialize-THTelemetry

## SYNOPSIS
Enable telemetry

## SYNTAX

```
Initialize-THTelemetry [[-ModuleName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Enable telemetry by creating a new telemetry client in the global telemetry store.
